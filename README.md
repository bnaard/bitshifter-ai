# Bitwise AI: MNIST Proof-of-Concept

This project explores **bit-operation-based neuronal models** for MNIST digit recognition, aiming to replace multiply-accumulate (MAC) operations with bitwise logic (shift, AND, OR, XOR, NOT) for efficient CPU execution.
