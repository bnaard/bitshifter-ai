# Research Papers

This directory contains all collected papers, citations, and summaries related to:
- Bit-operation-based neuronal models
- MAC-based models (for comparison)
- MNIST implementations

## Structure
- `paper_name.pdf`: Original paper
- `paper_name_summary.md`: Key points, relevance, and citations

## How to Add
1. Upload the PDF.
2. Create a Markdown summary with:
   - Title, authors, year
   - Key findings
   - Relevance to our project
   - Citation (BibTeX/APA)
