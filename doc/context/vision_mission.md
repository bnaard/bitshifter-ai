# Vision & Mission

## Vision
To democratize AI by developing alternative AI model algorithms and neuronal models that rely exclusively on bit operations (shift, roll, AND, OR, XOR, NOT). The goal is to create a proof-of-concept implementation that:
- Operates without specialized hardware (e.g., GPUs),
- Runs efficiently on general-purpose CPUs,
- Challenges the dominance of multiply-accumulate (MAC)-based models.

## Mission
1. **Research & Theoretical Foundation**: Investigate existing literature on bit-operation-based AI models and identify gaps.
2. **Proof-of-Concept Development**: Design and implement a neuronal model using only bit operations.
3. **Validation & Documentation**: Benchmark performance on a general-purpose CPU and publish findings as open-source code and a whitepaper.

## Key Decisions
- Use MNIST dataset for initial proof-of-concept (digit recognition).
- Implement first in Python (Jupyter Notebooks), then optimize in Rust.
- Include a MAC-based MNIST model as a baseline for benchmarking.
- Document all research results in a LaTeX-based scientific paper.
